▶ Level 1: Self

5 min | LOW | 0.6x

How Each SCOREs Dimension is Validated

S - Study Self-report: "Where do I learn about AI?"
C - Copy Self-report: "What tutorials have I followed?"
O - Output Self-report: "What have I deployed?"
R - Research Self-report: "What have I contributed?"
Es - Ethical Self-report: "How safely do I use AI?"

Evidence Required

  • None - honest self-reflection only
  • Mental inventory of your AI skills and experience
Best for: Personal development planning, identifying skill gaps, quick baseline check

▶ Level 2: Peer

30 min | MEDIUM | 0.8x

How Each SCOREs Dimension is Validated

S - Study Manager confirms learning activities, course completions, reading habits
C - Copy Peer reviews code/work samples showing replication of techniques
O - Output Manager confirms deployed projects, screenshots, demo recordings
R - Research Links to published work, shared methods, peer acknowledgment
Es - Ethical Manager attests to safe practices, no known incidents

Evidence Required

  • Work samples (GitHub repos, project docs)
  • Demo recordings or screenshots
  • Peer/colleague reviews
  • Manager sign-off on each dimension
Best for: Quarterly performance reviews, team calibration sessions, training allocation

▶ Level 3: Auto/Audit

1-2+ hrs | HIGH | 1.0x

How Each SCOREs Dimension is Validated

This level combines automated verification with full audit capabilities. Use automated methods for scalable validation, or conduct a full audit for high-stakes decisions. Both achieve 1.0x confidence.

S - Study Auto: LMS logs, RSS feeds
Audit: Live interview on concepts
C - Copy Auto: Git commits, Copilot logs
Audit: Live demo + code walkthrough
O - Output Auto: CI/CD + production metrics
Audit: ROI proof + testimonials
R - Research Auto: GitHub stats, citations
Audit: Papers, talks, recognition
Es - Ethical Auto: Audit logs, PII scans
Audit: Compliance records, post-mortems

Evidence Required

  • Automated: Git history, tool logs (Copilot, API calls), code review history, time savings data
  • Full Audit: Comprehensive portfolio with case studies, stakeholder testimonials, ROI documentation
  • Before/after comparisons, external audit reports, safety compliance records
Best for: Critical hiring decisions, promotion to senior AI roles, high-stakes project assignments, objective validation of claims
